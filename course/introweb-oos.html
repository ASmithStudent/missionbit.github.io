Intro to Computer Programming

Another program we are running this summer is a free immersive (6 week, 5 days/week, 7 hours/day) mobile game development class for 12 SF public high school students this summer. The class started on June 6th and ends on July 11. After two weeks of work on game design best practices, dev projects on sample mobile games and java tutorials, the class has now been split into 3 teams. By the end of the class, the students will have an Android game ready for the Google Play Store. The class has featured guest lecturers from SF mobile game companies such as Zynga, DeNA West, Glu, Chartboost and Pocket Gems.
