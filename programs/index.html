Free Coding Classes for Public School Students

Fall 2014

Intermediate Web Programming

The core work in this class will be using Ruby and Sinatra as well as some SQL and Javascript to build more complex web apps than the intro class. This class will dive deeper into core programming concepts through project based instruction.

Students will visit SF tech companies like Zynga, Pinterest and Airbnb, have the opportunity to be selected for paid internships and paid project work, receive and Mission Bit t-shirt.

  •  Mission High School – Tuesdays and Thursdays from 4pm-6pm. The first class is Tuesday, September 16th.

Intro Web Programming - High School

In the 'Intro Web Programming' class, students will learn HTML5, CSS3 and Javascript and some core programming concepts through project based instruction. Within the semester the class will build websites and interactive applications. At the end of the semester, the class is broken up into teams, each paired with an instructor and spends a few weeks building a final project. Projects in prior semesters have included HTML5 games, an interactive light show that change the hue of lights based on sound frequency, a browser-based drawing tool and a basic instant messaging app.

Intro Web Programming - Middle School

This class is designed specifically for middle school students. Students learn basic programming concepts leveraging Scratch, Bootstrap and learn basic HTML5, CSS3 and Javascript through project based instruction. Within the semester the class will build websites and interactive applications.
