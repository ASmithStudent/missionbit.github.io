Empowering youth with computer programming education and professional experiences

Spring 2015

High School Classes

High school students from any SFUSD high school are welcome to take Mission Bit classes free of charge.
All 2015 spring semester high school classes are hosted at Mission High School.
Students enrolled in 2015 spring semester high school classes will received priority consideration for 2015 summer paid internships.
All 2015 spring semester students will be invited to participate in class visits to a minimum of two SF Bay Area tech companies.
If you are an SFUSD teacher interested in partipcating in a 2015 spring semester class, please sign up as a student for the class that interests you.

Intermediate Web Programming w/ Ruby

Jesse Farmer, Lead Instructor

The core work in this class will be using Ruby and Sinatra as well as some SQL and Javascript to build more complex web apps than the Mission Bit intro class. This class will dive deeper into core programming concepts through project based instruction.

Meets: Tuesdays and Thurdays, 4pm - 6pm. The first class is Tuesday February 3rd 2015.

Intro Web Programming

Matt Hooker, Lead Instructor

In the 'Intro Web Programming' class, students will learn HTML5, CSS3 and Javascript and some core programming concepts through project based instruction. Within the semester the class will build websites and interactive applications. At the end of the semester, the class is broken up into teams, each paired with an instructor and spends a few weeks building a final project.

Meets: Tuesdays and Thursdays, 4pm - 6pm. The first class is Tuesday February 3rd 2015.
