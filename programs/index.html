Summer 2014

Mobile Game Development

Want to learn how to make games? There's still space to apply to our free 6 week full time mobile game development class from June 2 to July 11!

By the end of the course you'll have a playable mobile game, ready for release in the app store. You'll get to work with and learn from mentors who are professional software engineers, and get feedback from teams at game development studios.

Students should be in High School, entering High School in the Fall, or have graduated from High School no earlier than 2012. Students must also be able to get to class in San Francisco at 9am. Some programming or computer science experience is ideal. All students meeting the other requirements are eligible to apply, even if not a public school student.

Mission Bit will be teaching 10 classes in San Francisco during the Fall 2014 semester.

Intro Web Programming

In the 'Intro Web Programming' class, students will learn HTML5, CSS3 and Javascript and some core programming concepts through project based instruction. Within the semester the class will build websites and interactive applications. At the end of the semester, the class is broken up into teams, each paired with an instructor and spends a few weeks building a final project. Projects in prior semesters have included HTML5 games, an interactive light show that change the hue of lights based on sound frequency, a browser-based drawing tool and a basic instant messaging app.
