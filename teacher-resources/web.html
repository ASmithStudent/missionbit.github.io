Course Overview and Objectives

Course Description

This course is specifically designed for students with little to no coding experience. Students will learn HTML5, CSS3, and JavaScript, as well as some core programming concepts through project based instruction. Throughout the semester, students will build 3 websites: an adventure game, a portfolio, and a final project of their choice.

Topics include: HTML, CSS, JavaScript, UXD, and version control using git and Github.

Learning Goals

  • Technical skills:
    • Design, prototype, and implement a website
    • Describe the main features of HTML, CSS, and JavaScript
    • Use git and github for version control and code sharing (team / open source workflow - pull, push, add, commit)
    • Run python Server
    • Know basic terminal commands
    • Integrate JS into a web page
  • Other goals:
    • Students will work effectively on teams and review each other's code.
    • Students will feel like they belong in tech
    • Students will have insights into what itâ€™s like to work in the tech industry (pros and cons)
    • Students will be able to pitch their project idea
    • Students will develop public speaking skills
  • Projects:
    1. P1: Adventure Game
    2. P2: Portfolio
    3. Final Project: Website - Open theme
