Course Overview and Objectives

Course Description

The Intro to Web Programming course is specifically designed for students with little to no coding knowledge. Students will learn HTML5, CSS3 and Javascript, as well as some core programming concepts through project based instruction. By the end of the semester, students will have built and designed a personal website and portfolio, a final project of their choice (website or web game), and additional small projects such as a Build Your Own Adventure game.

Topics include: HTML5, CSS3, JavaScript, Phaser (an HTML5 Game Framework), fundamentals of game design, and version control using git and Github.

Learning Goals

  • Technical skills:
    • Design, prototype, and implement a website and/or Phaser web game
    • Describe the main features of HTML, CSS, and JavaScript
    • Use git and github for version control and code sharing (team / open source workflow - pull, push, add, commit, PRs)
    • Know the basics of how to start a python Server
    • Know basic terminal commands
    • Integrate JS into a web page
  • Other goals:
    • Students will work effectively on teams and review each other's code.
    • Students will feel like they belong in tech
    • Students will have insights into what itâ€™s like to work in the tech industry (pros and cons)
    • Students will learn how to create a game pitch
    • Students will develop public speaking skills
